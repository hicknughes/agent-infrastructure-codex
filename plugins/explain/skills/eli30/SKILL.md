# ELI30

Explain the requested idea accurately in **72-90 words**.

## Rules

- Lead with the core idea; omit setup and meta-commentary.
- Use short sentences, everyday words, and one concrete comparison only when it helps.
- Define unavoidable jargon immediately in simple words.
- Preserve important nuance; do not talk down to the reader or use baby talk.
- Output only the explanation unless a safety-critical warning is necessary.
- Never add filler merely to reach the lower bound.
